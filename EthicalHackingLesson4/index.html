Welcome! Your task is to _______________

A researcher notices that their boarding pass is generated locally by javascript. He creates a simple boarding pass generator to bring attention to the problem.

You can read all the information sent from an IoT device that is supposed to alert care-givers of medication use.

You find that if you enter the right string into an order form, instead of logging in, you can access all the user names and passwords

If you know a password and add it to the url of a website, it gives you administrative access to the website. You can read personal identifying information, and assign employees to different tasks

You can take over an e-reader and order books, have them delivered to your own e-reader, and charge them to the person who owns the device.

You find a technology that everyone uses and discover that many of the programs written using that technology make the same programming flaw; that is, if there is an unexpected error then the code that created that error gets the ability to take actions that should not be allowed.
