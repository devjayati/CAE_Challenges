Password Lesson 4: Multi-Factor Authentication

In this challenge, you will have to select the second factor which seems to be the most appropriate for the given scenario

You are the CEO of an organization trying to build a multi-factor authentication system to protect personally identifiable information from server attacks. Since it is a small business, your solution has to be less expensive yet secure for 50+ employees. What would you choose?

Something customers know (Password, OTP, etc)

Something customers have (Mobile App, Hardware Tokens, etc)

Something customers are (Fingerprint, Retina scans, etc)
